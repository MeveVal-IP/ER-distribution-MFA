MFA summary: exponential vs stationary phase

What this summary contains:
This file is a compact overview of marker frequency analysis (MFA) outputs from the two full reports (exponential phase and stationary phase). It aggregates the per-genome tables from those reports and shows four summary plots:

  • Row 1: chromosome-to-secondary replicon ori/ori and ter/ter ratios (EXP and STAT).
  • Row 2: replication slope magnitudes (EXP and STAT) for each replichore and replicon.

Missing values: any entries recorded as NA in the source tables are not plotted here. Each row uses the same y-axis scale for EXP vs STAT to enable direct comparison.

Excluded: Leptospira and Deinococcus are omitted from all plots.

Row 1: ori/ori and ter/ter ratios

Row 1 (ratios): colors indicate replicon order within each genome: green = chromosome-to-second replicon, yellow = chromosome-to-third replicon. Shapes: ■ = ori/ori, ● = ter/ter. y-axis: 0.5 to 2.
